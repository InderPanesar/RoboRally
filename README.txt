                                                            ==============================================================
                                                            =  This is a simplified version of Robo Rally! The Robots    =
                                                            =  begin by looking north. They have 5 commands.             =
                                                            =  1. (F) Forward one space                                  =
                                                            =  2. (B) Back one space                                     = 
                                                            =  3. (L) Rotate 90 Degrees Anti-Clockwise                   =
                                                            =  4. (R) Rotate 90 Degrees Clockwise                        =
                                                            =  5. (U) Rotate 180 Degrees                                 =
							    =  6. (W) Wait                                               =
                                                            = ============================================================
                                                            =  You cannot repeat command: FFWWL, UUWLF                   =
                                                            =  The Robots will play in player order in a token based     =
                                                            =  system!                                                   =
                                                            =  Robots will push a robot if they move into a place where  =
                                                            =  a Robot already is.                                       =
							    =  If the Robot moves outside the board the Robot is destroy-=
							    =  ed and returns to it's starting position                  =
                                                            =  The robots start with N Health                            =
                                                            =  Robots with N Health can make N moves.                    =
                                                            ==============================================================
							    =  Board have the following locations:                       =
							    =  Empty (.) - Nothing exists                                =
                                                            =  Flags (1-4) - Used to track need to get all flags in      =
                                                            =  order to win                                              = 
                                                            =  Pits (x) - Destroy your robot on entry.                   = 
                                                            =  Gears (+,-) - Rotate your robot 90 Degrees Clockwise and  =
                                                            =  Anti-clockwise                                            = 
                                                            =  Lasers ( [, ], ( ,) )- Destroy your robot if your robot   =
                                                            =  is within the beam.                                       =
                                                            =  Conveyor Belt (v, >, <, ^) - Moves your belt in a         =
							    =  specific direction                                        =
                                                            ==============================================================
							    =  Created in Java, using Eclipse. JavaFX was used in the UI =
                                                            =                 JUnit 5 used for the testing               =
